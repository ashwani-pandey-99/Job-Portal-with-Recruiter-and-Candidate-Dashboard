# 🚀 Job Portal with Recruiter & Candidate Dashboard

A modern and user-friendly job portal that connects recruiters with talented candidates. Built using the MERN stack, this platform provides a seamless experience for job searching, application tracking, and recruitment management.

---

## 🌟 Features

### 👨‍💻 Candidate

* Search and filter jobs 
* Apply to jobs in one click
* Track application status
* Build and manage profile

### 🏢 Recruiter

* Post and manage job listings
* View and shortlist candidates
* Manage applications efficiently
* Dashboard for hiring insights

## 🛠️ Tech Stack

* **Frontend:** React, Tailwind CSS, Vite
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT
* **Deployment:** Vercel / Render

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 2️⃣ Install dependencies

```bash
cd backend
npm install

cd ../frontend
npm install
```

### 3️⃣ Run the project

```bash

# backend

npm run dev

# frontend

npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
CLOUDINARY_API_KEY=your_key
```
